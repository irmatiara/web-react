# Klaim:
1. Penggunaan Props (20 Point) - Props digunakan untuk mengirim data artikel dari parent component (index.jsx) ke child component (ArtikelPopuler) agar artikel dapat ditampilkan secara dinamis dan kode tetap terstruktur.
2. Penggunaan Event (40 Point) - Event digunakan dengan onClick untuk mengatur menu aktif pada navbar dan onChange pada form kontak, di mana form akan di-reset setelah pengguna menekan tombol kirim tanpa perlu me-refresh halaman.
3. Penggunaan React Router untuk Navigasi Halaman (20 Point) - React Router digunakan untuk mengatur navigasi halaman dan menandai menu yang aktif berdasarkan URL, sehingga memudahkan pengguna dalam mengetahui halaman yang sedang diakses.
4. Penggunaan FrameWork Tailwind (20-40 Point) - Tailwind CSS membantu membuat antarmuka yang rapi dan terstruktur tanpa menulis CSS tambahan.
