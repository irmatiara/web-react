Awal Mula Website

Bagaimana web lahir dan berubah jadi dunia digital besar seperti sekarang?

1989 — Ide Website Lahir

Sir Tim Berners-Lee punya visi: bagaimana jika kita bisa saling berbagi informasi dengan cara yang lebih mudah antar komputer di seluruh dunia? Dari sinilah konsep World Wide Web (WWW) muncul sebagai solusi.

1991 — Website Pertama di Dunia

Website pertama online dengan alamat: info.cern.ch Isinya? Penjelasan tentang apa itu World Wide Web! Sederhana banget, hanya teks hitam putih tanpa gambar.

1993 — Browser dengan Tampilan Gambar

Mosaic hadir dengan inovasi besar: menampilkan gambar di dalam halaman! Internet mulai terasa seperti “dunia baru yang bisa dilihat”.

1995 — HTML 2.0 dan Web Modern Dimulai

Muncullah standar HTML. Website mulai memiliki formulir, hyperlink lebih teratur, dan tampilan tidak monoton lagi.

2000-an — Web Jadi Sosial

Era Web 2.0: orang tak lagi hanya membaca — tapi bisa menulis, mengupload foto, berkomentar, dan berbagi. Muncul platform seperti Facebook, YouTube, dan Wikipedia.

2020-an — Web Ada di Mana-Mana

Smartphone. IoT. AI. VR. Website kini bukan cuma halaman informasi — tapi bagian dari hidup kita setiap hari.

Website pertama masih aktif sampai sekarang, bisa kamu kunjungi kapan pun!
